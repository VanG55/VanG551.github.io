## Введение

NekoRay - это многофункциональный VPN-клиент для Android, Windows и Linux, предназначенный для работы с современными прокси-протоколами семейства v2Ray. Приложение ориентировано на продвинутых пользователей и предоставляет расширенные возможности настройки VPN-подключений.

> **Обратите внимание**

> Приложение не предоставляет VPN‑серверы само по себе: пользователь должен настроить сервер самостоятельно или получить конфигурацию у провайдера. Получить конфигурацию можно в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

### Поддерживаемые протоколы

- VLESS
- VMess
- Trojan
- Shadowsocks
- Hysteria / Hysteria2
- SOCKS / HTTP

### Подключение через Nekoray

### Шаг 1. Установите приложение

1. Откройте страницу релизов в репозитории на GitHub: [NekoBox Releases](https://github.com/MatsuriDayo/NekoBoxForAndroid/releases).
2. Загрузите .apk файл NekoRay последней версии (например, ссылка на версию 1.3.8 выше).
3. Откройте скачанный файл, разрешите установку из этого источника и установите приложение.

> **Подсказка:** В целях безопасности рекомендуется скачивать приложение только с официального репозитория GitHub.

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

1. Скопируйте вашу конфигурацию (ссылку на подписку).
2. Откройте боковое меню приложения и перейдите во вкладку **"Группы"**.
3. Нажмите на плюс «+» в правом верхнем углу.
4. Укажите удобное имя группы.
5. Выберите тип группы **"Подписка"**.
6. Вставьте ваш ключ в поле **"Ссылка для подписки"**.
7. Пролистайте ниже до пункта **"Автоматическое обновление"** и включите его.
8. Сохраните изменения, нажав на «✓» в правом верхнем углу.

### Шаг 3. Подключитесь и пользуйтесь

1. Вернитесь на главную страницу приложения.
2. Выберите локацию.
3. Нажмите на кнопку подключения в нижней области экрана.
