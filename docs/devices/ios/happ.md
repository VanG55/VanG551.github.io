# Подключение к VPN через Happ на iPhone и iPad (VLESS)

В данном руководстве описывается процесс загрузки, установки и настройки приложения Happ на iPhone и iPad с iOS или iPadOS для удобного использования VPN через протокол VLESS.

- Требуется iOS 15.0 или новее.
- Загрузить в App Store → [Happ в App Store](https://apps.apple.com/us/app/happ-proxy-utility/id6504287215)

## Что такое Happ

Happ - кроссплатформенное приложение для безопасного проксирования интернет-трафика на базе Xray (форка V2Ray). Оно предназначено для создания защищённых соединений с VPN-серверами по современным протоколам. У данного приложения интуитивный интерфейс и гибкие настройки, которые делают Happ удобным для любого пользователя.

### Основные возможности Happ:

- Настройка локальных прокси-серверов на устройстве по гибким правилам маршрутизации трафика.
- Поддержка множества современных протоколов передачи данных (VLESS [с опцией Reality], VMess, Trojan, Shadowsocks, Socks).
- Гарантия конфиденциальности — приложение не собирает данные о сетевой активности и не передаёт их на сторонние сервера.
- Отсутствие собственных VPN-серверов — Happ не продаёт доступ к серверам, пользователь должен настроить их самостоятельно или получить готовую конфигурацию от провайдера.

### Принцип работы. 
Happ фактически запускает на устройстве локальный прокси (обычно HTTP/SOCKS5), через который перенаправляет весь интернет-трафик на удалённый сервер. Для этого пользователь импортирует в приложение конфигурацию сервера в виде ссылки или подписки (например, vless:// или vmess:// с настройками). Поскольку Happ построен на ядре Xray, соединение устанавливается по выбранному протоколу с шифрованием, обеспечивая защиту трафика и обфускацию (скрытие) содержимого. В настройках можно задать правила маршрутизации: например, выводить через VPN лишь трафик определённых сайтов или стран. При этом само приложение не предоставляет VPN-сервисы — это лишь инструмент для подключения к уже существующим серверам

## Как подключиться через Happ

> **Обратите внимание**

> Приложение не предоставляет VPN‑серверы само по себе: пользователь должен настроить сервер самостоятельно или получить конфигурацию у провайдера. Получить конфигурацию можно в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

## Шаг 1. Установите приложение

1. Откройте страницу в App Store: [Happ в App Store](https://apps.apple.com/us/app/happ-proxy-utility/id6504287215) и установите приложение. 

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

В Happ нажмите кнопку «+» и выберите «Из буфера обмена». Приложение автоматически обнаружит скопированную ссылку и предложит добавить новую конфигурацию сервера.

> При первом импорте Happ может запросить доступ к буферу обмена — согласитесь с запросом, чтобы вставить конфигурацию

### Шаг 3. Подключитесь и пользуйтесь

В списке серверов выберите подходящую локацию и нажмите большую кнопку включения. 

После этого Happ установит VPN-соединение с выбранным сервером, и интернет-трафик вашего устройства пойдёт через него. Об активном VPN-подключении будет сигнализировать значок ключа или замка в строке состояния.
