# Подключение к VPN через NekoRay на Linux (VLESS)

В данном руководстве описывается процесс загрузки, установки и настройки приложения NekoRay на Linux (пример на Ubuntu 24.04 LTS) для удобного использования VPN через протокол VLESS.

- **Официальный сайт проекта:** https://github.com/MatsuriDayo/nekoray
- **Релизы в GitHub:** https://github.com/MatsuriDayo/nekoray/releases
- **Репозиторий GitHub:** https://github.com/MatsuriDayo/nekoray

## Введение

NekoRay — это клиент для подключения к VPN для систем Android, Windows и Linux, предназначенный для безопасного проксирования интернет‑трафика через протоколы семейства v2Ray, таких как VLESS, VMess, Trojan, Shadowsocks, Hysteria и др.

> **Обратите внимание**

> В зависимости от вашего дистрибутива Linux этапы установки и настройки могут отличаться. В этом руководстве рассматривается Ubuntu 24.04 LTS.

### Шаг 1. Установите приложение

#### Установка с помощью скрипта (автоматическая)
<details>
<summary>Показать / скрыть инструкции скрипта</summary>

Для большинства дистрибутивов можно использовать готовый скрипт для автоматической установки NekoRay и добавления ярлыка в меню приложений.

1. Установите зависимости (Debian/Ubuntu):

```
sudo apt update && sudo apt install -y build-essential libfontconfig1 libqt5network5 libqt5widgets5 libqt5x11extras5 libqt5gui5
```

2. Запустите установочный скрипт (от имени пользователя):

```
wget -qO- https://raw.githubusercontent.com/ohmydevops/nekoray-installer/main/installer.sh | bash
```

После завершения скрипта ярлык появится в меню приложений.

</details>

#### Установка на Debian/Ubuntu из .deb
<details>
<summary>Показать / скрыть инструкцию для .deb</summary>

1. Скачайте .deb пакет релиза, например: `nekoray-4.0.1-2024-12-12-debian-x64.deb` с релизов GitHub.
2. Установите пакет через терминал:

```
sudo apt install ./nekoray-4.0.1-2024-12-12-debian-x64.deb
```

3. После установки приложение появится в меню приложений.

</details>

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

1. Скопируйте вашу конфигурацию (ссылку на подписку).
2. В приложении выберите **"Сервер" → "Добавить из буфера обмена"** или вставьте сочетанием Ctrl+V.
3. В появившемся окне выберите **"Как подписку (создать новую группу)"**.

### Шаг 3. Настройте приложение перед подключением

1. Перейдите в **Настройки → Настройки TUN‑режима**.
2. Включите **"Режим белого списка"** при необходимости.
3. В **Настройки → Основные настройки → Подписка** включите **"Автоматическое обновление"**.

### Шаг 4. Подключитесь и пользуйтесь

1. Вернитесь на главную страницу NekoRay.
2. Откройте вкладку с подпиской.
3. Выберите нужный сервер и нажмите **"Запустить"**.
4. Включите **"Режим TUN"** для полного туннелирования трафика.

## Режимы работы: TUN vs Системный прокси

**TUN режим**
- Работает для всех программ в системе.
- Создаёт виртуальный сетевой адаптер и перехватывает весь трафик.
- Требует прав администратора.

**Системный прокси**
- Работает только для приложений, учитывающих системный прокси.
- Менее требователен к правам, запускается быстрее.
