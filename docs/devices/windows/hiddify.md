# Подключение к VPN через Hiddify на Windows (VLESS)

В данном руководстве описывается процесс загрузки, установки и настройки приложения Hiddify на ПК или ноутбуке с Windows для удобного использования VPN через протокол VLESS.

- Требуется Windows 10 или новее.
- Загрузить (2.5.7, Стабильная версия) → [Скачать Hiddify 2.5.7](https://github.com/hiddify/hiddify-app/releases/download/v2.5.7/Hiddify-Windows-Setup-x64.exe)
- Репозиторий GitHub → [hiddify/hiddify-app](https://github.com/hiddify/hiddify-app)

## Введение

Hiddify - это универсальный VPN-клиент для различных операционных систем, предназначенный для безопасного и стабильного доступа к интернету с использованием современных прокси-протоколов семейства v2Ray. Приложение подходит как для обычных пользователей, так и для продвинутых администраторов VPN-сервисов.

Поддерживаемые протоколы:
- VLESS
- VMess
- Trojan
- Shadowsocks
- Hysteria и Hysteria2

> **Обратите внимание**

> Приложение не предоставляет VPN‑серверы само по себе: пользователь должен настроить сервер самостоятельно или получить конфигурацию у провайдера. Получить конфигурацию можно в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

## Как работает Hiddify

Принцип работы Hiddify состоит из нескольких этапов:

1. Пользователь получает конфигурацию подключения (чаще всего в виде ссылки-подписки).
2. Конфигурация импортируется в приложение Hiddify.
3. Приложение устанавливает защищённое соединение с VPN-сервером.
4. Весь интернет-трафик устройства проходит через зашифрованный туннель.

Hiddify автоматически:

- выбирает доступные серверы;
- поддерживает обновление подписок;
- позволяет быстро переключаться между прокси-узлами.

## Как подключиться к VPN через Hiddify

### Шаг 1. Установите приложение с правами администратора

1. Загрузите приложение Hiddify для Windows по ссылке с GitHub: [репозиторий релизов](https://github.com/hiddify/hiddify-app).
2. Запустите .EXE файл установщика с правами администратора и пройдите все этапы установки.
3. На рабочем столе появится ярлык Hiddify, кликните по нему правой кнопкой мыши и перейдите в **Свойства → Совместимость**.
4. Установите галочку напротив пункта **"Запускать эту программу от имени администратора"**.

> **Подсказка:** В целях безопасности рекомендуется скачивать приложение только с официального репозитория GitHub.

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

1. Скопируйте вашу конфигурацию (ссылку на подписку).
2. Нажмите на плюс «+» в правом верхнем углу главной страницы Hiddify.
3. Выберите **"Добавить из буфера обмена"** и дождитесь сообщения об успешном импорте профиля.

### Шаг 3. Настройте приложение перед подключением

1. Откройте вкладку **"Параметры конфигурации"** в боковом меню и установите (проверьте) необходимые значения.

> **Обратите внимание:** Без активного режима VPN соединение может распространяться не на все приложения в системе.

### Шаг 4. Подключитесь и пользуйтесь

1. Вернитесь на главную страницу Hiddify.
2. Нажмите большую кнопку подключения по центру.
