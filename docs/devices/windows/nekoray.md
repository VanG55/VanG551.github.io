# Подключение к VPN через NekoRay на Windows (VLESS)

В данном руководстве описывается процесс загрузки, установки и настройки приложения NekoRay на ПК или ноутбуке с Windows для удобного использования VPN через протокол VLESS.

- Требуется Windows 8 или новее.
- Загрузить (v4.0.1) → [скачать NekoRay 4.0.1 (zip)](https://github.com/MatsuriDayo/nekoray/releases/download/4.0.1/nekoray-4.0.1-2024-12-12-windows64.zip)
- Репозиторий GitHub → [MatsuriDayo/nekoray](https://github.com/MatsuriDayo/nekoray)

## Введение

NekoRay — это клиент для подключения к VPN для систем Android, Windows и Linux, предназначенный для безопасного проксирования интернет‑трафика через протоколы семейства v2Ray, таких как VLESS, VMess, Trojan, Shadowsocks, Hysteria и др.

> **Обратите внимание**

> Приложение не предоставляет VPN‑серверы само по себе: пользователь должен настроить сервер самостоятельно или приобрести конфигурацию у провайдера. Получить конфигурацию можно в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

### Шаг 1. Установите приложение с правами администратора

1. Загрузите архив с приложением NekoRay для Windows с релизов на GitHub: [NekoRay Releases](https://github.com/MatsuriDayo/nekoray).
2. Распакуйте архив в удобное место на диске (избегайте кириллических символов в пути).
3. Откройте папку с NekoRay и кликните по `nekobox.exe` правой кнопкой мыши → **Свойства → Совместимость**.
4. Установите галочку **"Запускать эту программу от имени администратора"**.
5. Создайте ярлык `nekobox.exe` на рабочем столе для удобства.
6. Запустите `nekobox.exe` с правами администратора.

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

1. Скопируйте вашу конфигурацию (ссылку на подписку).
2. Нажмите **"Сервер" → "Добавить из буфера обмена"** или используйте Ctrl+V.
3. В появившемся окне выберите **"Как подписку (создать новую группу)"**.

### Шаг 3. Настройте приложение перед подключением

1. Откройте **Настройки → Настройки TUN‑режима**.
2. Поставьте галочку в чекбоксе **"Режим белого списка"** в левом нижнем углу.
3. Нажмите **ОК** и закройте окно.
4. Перейдите в **Настройки → Основные настройки → Подписка**.
5. Включите **"Автоматическое обновление"**.
6. Нажмите **ОК** и закройте окно.

### Шаг 4. Подключитесь и пользуйтесь

1. Вернитесь на главную страницу NekoRay.
2. Откройте вкладку с подпиской.
3. Выберите нужный сервер: кликните правой кнопкой мыши по строке с сервером → **"Запустить"**.
4. Поставьте галочку в чекбоксе **"Режим TUN"**.

## В чем различие режимов работы?

**TUN режим (VPN‑режим)**
- Работает для всех программ на компьютере.
- Создаёт виртуальный сетевой адаптер и перехватывает весь системный трафик.
- Требует прав администратора и может занимать до 30 секунд при активации.

**Системный прокси**
- Работает только для приложений, учитывающих системные настройки прокси.
- Не требует особых прав для запуска и переключается быстрее.
