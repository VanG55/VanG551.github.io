# Подключение к VPN через v2RayN на Windows (VLESS)

В данном руководстве описывается процесс загрузки, установки и настройки приложения v2RayN на ПК или ноутбуке с Windows для удобного использования VPN через протокол VLESS.

- Требуется 64‑разрядная Windows 8 или новее.
- Загрузить (v7.10.4) → [скачать v2rayN 7.10.4](https://github.com/2dust/v2rayN/releases/download/7.10.4/v2rayN-windows-64.zip)
- Репозиторий GitHub → [2dust/v2rayN](https://github.com/2dust/v2rayN)
- Загрузить Microsoft Windows Desktop Runtime .NET → [скачать .NET](https://dotnet.microsoft.com/ru-ru/download/dotnet/9.0)

## Введение

v2RayN — это мощный VPN-клиент для операционных систем Windows, предназначенный для подключения к прокси- и VPN-серверам на базе протоколов семейства v2Ray. Приложение ориентировано на продвинутых пользователей и предоставляет расширенные возможности управления VPN-соединениями.

> **Обратите внимание**

> Приложение не предоставляет VPN‑серверы само по себе: пользователь должен настроить сервер самостоятельно или получить конфигурацию у провайдера. Получить конфигурацию можно в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

### Поддерживаемые протоколы

- VLESS
- VMess
- Trojan
- Shadowsocks
- Hysteria и Hysteria2

## Как подключиться к VPN через v2RayN

### Шаг 1. Установите приложение с правами администратора

1. Загрузите архив v2rayN‑windows‑64.zip с релизов на GitHub: [страница релизов](https://github.com/2dust/v2rayN).
2. Распакуйте архив в удобное место на диске (избегайте кириллических символов в пути).
3. Откройте папку с v2rayN и кликните по v2rayN.exe правой кнопкой мыши → **Свойства → Совместимость**.
4. Установите галочку **"Запускать эту программу от имени администратора"**.
5. Для удобства создайте ярлык v2rayN.exe на рабочем столе.
6. Запустите v2rayN.exe с правами администратора.

> **Обратите внимание:** При первом запуске может потребоваться установка Microsoft Windows Desktop Runtime (.NET). Согласитесь и установите среду выполнения с официального сайта Microsoft.

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

1. Скопируйте вашу конфигурацию (ссылку на подписку).
2. Нажмите **"Сервер" → "Импорт массива URL из буфера обмена"** или вставьте сочетанием клавиш Ctrl+V.
3. Для отображения списка доступных локаций нажмите **"Группа подписки" → "Обновить подписку без прокси"**.

Совет: сменить язык приложения можно через меню **⋮** в правом верхнем углу.

### Шаг 3. Подключитесь и пользуйтесь

1. Выберите локацию: кликните правой кнопкой мыши по строке с сервером → **"Установить как активный сервер"**.
2. Активируйте переключатель **"Режим VPN"** внизу приложения.

> **Подсказка:** Если при запуске режима VPN (TUN) возникают ошибки, попробуйте режим **"Системный прокси"** (в меню справа) — в этом режиме не все приложения будут использовать VPN.
