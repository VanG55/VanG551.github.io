# Подключение к VPN через v2RayTun на Windows (VLESS)

В данном руководстве описывается процесс загрузки, установки и настройки приложения v2RayTun на ПК или ноутбуке с Windows для удобного использования VPN через протокол VLESS.

- Требуется Windows 8.1 или новее.
- Загрузить последнюю версию (.exe) → [Скачать v2RayTun](https://storage.v2raytun.com/v2RayTun_Setup.exe)

## Введение

v2RayTun - это многофункциональный VPN/прокси‑клиент на базе ядра Xray (v2Ray), доступный для Android, iOS, Windows и других платформ. Приложение позволяет безопасно проксировать интернет‑трафик через серверы провайдера, поддерживает современные протоколы семейства v2Ray и не хранит пользовательских данных (все настройки и логи остаются на устройстве). v2RayTun не предоставляет собственных VPN‑серверов - необходимо самостоятельно иметь или приобрести доступ к конфигурации сервера.

> **Подсказка**

> Получить конфигурацию можно в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

## Основные возможности
- Проксирование трафика. v2RayTun перенаправляет сетевой трафик через удалённый сервер (VPN/прокси), обеспечивая обход блокировок и шифрование соединения
- Поддержка протоколов V2Ray. Клиент работает с протоколами VLESS, VMess, Trojan, Shadowsocks и SOCKS, а также поддерживает современный вариант Reality (XTLS) для повышенной безопасности.
- Разнообразные шифрования. Доступна гибкая настройка криптографической защиты: алгоритмы AES-128/192/256-GCM и ChaCha20-IETF (Poly1305). v2RayTun использует расширенные схемы шифрования Xray, не собирает логи и гарантирует конфиденциальность передачи данных.
- Гибкая маршрутизация. В настройках можно указать, какие приложения или подключения должны идти через прокси, а какие — напрямую в интернет. Это позволяет создавать собственные правила (split‑tunneling) для оптимизации работы.
- Удобный импорт конфигураций. v2RayTun позволяет быстро импортировать готовые настройки из разных источников: сканировать QR‑код, вставлять ссылку или текстовую конфигурацию из буфера обмена, а также вводить параметры вручную.
- После импорта нужный сервер тут же появляется в списке доступных.
- Приватность и производительность. Приложение не ограничивает скорость соединения и не собирает данные о пользователях. Все параметры подключения хранятся локально, а сам клиент обеспечивает шифрование трафика «от устройства до сервера».

## Установка и настройка
Чтобы подключиться к VPN через v2RayTun, выполните несколько простых шагов:

### Шаг 1. Установите приложение с правами администратора

1. Загрузите установщик v2RayTun: [скачать .EXE](https://storage.v2raytun.com/v2RayTun_Setup.exe).
2. Запустите .EXE файл установщика с правами администратора и пройдите этапы установки.
3. На рабочем столе появится ярлык v2RayTun. Кликните по нему правой кнопкой мыши и перейдите в **Свойства → Совместимость**.
4. Установите галочку напротив пункта **"Запускать эту программу от имени администратора"**.

> **Подсказка:** Запуск от администратора необходим для корректной работы системного проксирования и правил маршрутизации.

### Шаг 2. Установите конфигурацию в приложение

> **Подсказка:** Бесплатную конфигурацию можно получить в Telegram‑боте [@VangVPN_bot](https://t.me/VangVPN_bot).

1. Скопируйте вашу конфигурацию (ссылку на подписку).
2. Нажмите на плюс «+» в правом верхнем углу вкладки **"Подключение"**.
3. Выберите **"Импорт из буфера обмена"**, после чего подписка со списком серверов добавится автоматически.

### Шаг 3. Настройте приложение перед подключением

1. Перейдите в нижнюю вкладку **"Настройки"**.
2. Откройте раздел **"Настройки трафика"**.
3. Проверьте значение DNS.
4. Установите режим **"Туннель"**.

> **Обратите внимание:** Без активного режима туннелирования VPN‑соединение может распространяться не на все приложения в системе.

### Шаг 4. Подключитесь и пользуйтесь

1. Выберите локацию.
2. Нажмите большую кнопку подключения по центру.
