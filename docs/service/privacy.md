# Политика конфиденциальности

Настоящая Политика конфиденциальности (далее — «Политика») описывает, какие данные собираются и обрабатываются при использовании VPN‑сервиса (далее — «Сервис»), и как обеспечивается их защита. Используя Сервис, вы подтверждаете, что ознакомились и согласны с настоящей Политикой.

## 1. Сбор информации

1.1. При использовании Сервиса мы обрабатываем минимально необходимый набор данных:

- Telegram ID пользователя — для идентификации и предоставления доступа к Сервису.
- Данные о платежах — обрабатываются через защищённые платёжные системы (мы не храним данные банковских карт).
- Технические данные (временные логи подключения) — сохраняются не более 24 часов и используются исключительно для обеспечения работоспособности Сервиса.

1.2. Мы не собираем и не храним: историю посещённых сайтов; содержимое интернет‑трафика; IP‑адреса, с которых подключается Пользователь (за исключением временных логов в рамках работы Сервиса).

## 2. Использование информации

2.1. Собранные данные используются исключительно для: предоставления доступа к Сервису; обработки и подтверждения платежей; обеспечения безопасности и стабильной работы Сервиса; выполнения требований законодательства.

## 3. Передача информации третьим лицам

3.1. Данные Пользователей не передаются третьим лицам, за исключением случаев, когда это необходимо: для выполнения требований законодательства; для обработки платежей (через надёжные платёжные системы).

## 4. Хранение и защита информации

4.1. Мы применяем технические и организационные меры для защиты данных от несанкционированного доступа.

4.2. Временные технические логи автоматически удаляются не позднее 72 часов с момента их создания.

4.3. Платёжные данные хранятся исключительно у платёжных операторов.

## 5. Права пользователя

5.1. Пользователь вправе: запросить удаление своего Telegram ID из базы данных Сервиса; прекратить использование Сервиса в любое время; обратиться в Администрацию для уточнения вопросов, связанных с обработкой данных.

## 6. Изменения в Политике

6.1. Мы вправе изменять настоящую Политику.

6.2. Обо всех значимых изменениях мы уведомляем Пользователей за 7 дней до вступления изменений в силу.

6.3. Продолжение использования Сервиса означает согласие с новой редакцией Политики.

Используя Сервис, вы подтверждаете, что прочитали, поняли и приняли условия данной Политики.
